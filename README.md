# Superstore-Sales
The Superstore dataset includes sales records of an office supply retailer, containing fields Order Date, Ship Mode, Segment, Category, Sub-Category, Region, Sales, Quantity, Discount, and Profit. Using Power BI, dimensions were analyzed to create comprehensive overview of business performance across multiple dimensions: product, geography &amp; time.

Key Insights and Features :-

1. Sales Performance Analysis: Tracks overall sales and profit trends over time with month-over-month and year-over-year comparisons.
2. Regional Breakdown: Visualizes top-performing regions and states using interactive map visuals, helping identify geographical growth opportunities.
3. Category & Sub-Category Insights: Highlights profitable product lines and areas with declining margins.
4. Customer Segmentation: Shows revenue distribution by customer segment (Consumer, Corporate, and Home Office).
5. Profitability Indicators: Uses DAX measures to compute KPIs like profit margin %, average order value, and discount impact on profits.
6. Dynamic Filters: Enables users to filter by category, region, or segment for tailored insights.

Tools and Technologies :-

Power BI: Dashboard creation and data visualization.
Excel: Data cleaning and preparation.
